Eloick Mickisz
17 ans
Terminal STI2D
Eleve de terminal STI2D, j'ai une passion pour l'informatique notamment le developpement.
J'ai commencé à m'interesser à l'informatique par le biais du javascript en réalisant un bot sur discord. Discord étant une application web permettant de discuter, appeler ses amis. Ce bot permis de rajouter des fonctionnalités comme par exemple : mettre une personne en muet pendant une certaine durer, avant cela il fallait effectuer plusieurs actions pour le faire et autant d'action pour l'enlever, alors qu'avec le bot crée il suffaisait d'entrer dans le chat une commande suvit du nom de la personne et du temps.
Par la suite, avec l'aide du moteur Construc 2, j'ai crée un petit jeu de plateforme. Il s'agit d'un dinosaure qui doit récolter le plus de pièce tout en évitant les ennemis dans différents niveaux. Ce jeu est disposible dans la section "Un petit jeu". Je l'avais fait tester à quelques amis qu'ils puissent me remonter les bugs et que je puisse les corriger.
J'ai crée aussi ce site Web en utilissant les languages CSS et HTML 5. Pour programmer, j'ai pris le logiciel Visual Studio Code. J'avais commencé avec Atom mais celui çi avait moins de fonctionnalités que Visual Studio Code qui est aussi plus personalisable.

En terminal, j'ai pris la spécialité Système de l'Information et du Numérique
