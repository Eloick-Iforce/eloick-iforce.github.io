Terminal STI2D
Eleve de terminal STI2D, j'ai une passion pour l'informatique notamment le developpement.
J'ai commencé à m'interesser à l'informatique par le biais du javascript en réalisant un bot sur discord. Discord étant une application web permettant de discuter, appeler ses amis. Ce bot permet de rajouter des fonctionnalités comme par exemple : mettre une personne en muet pendant un certain temps, avant cela il fallait effectuer plusieurs actions pour le faire et autant pour l'enlever, alors qu'avec le bot crée il suffisait d'entrer dans le chat, le nom de la personne à mettre en muet, le temps, avec une seule commande.
Par la suite, avec l'aide du moteur Construc 2, j'ai crée un petit jeu de plateforme. Il s'agit d'un dinosaure qui doit récolter le plus de pièces tout en évitant les ennemis dans différents niveaux. Ce jeu est disponible dans la section "Un petit jeu". Je l'avais fait tester à quelques amis afin qu'ils puissent me remonter les bugs et que je les corrige.
J'ai crée aussi ce site Web en utilisant les languages CSS et HTML 5. Pour programmer, j'ai pris le logiciel Visual Studio Code. J'avais commencé avec Atom mais celui çi avait moins de fonctionnalités que Visual Studio Code qui est aussi plus personalisable.
